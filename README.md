# BrighterDayBouquet
This is an e-commerce website for selling flowers to local customers (no delivery options planned for release)

## Technologies Used:
### Backend:
  * ASP.NET Core 1.0.0 - Including:
    * MVC Core
    * Identity
    * Entity Framework Core for SQL Server
    
### Frontend:
  * Angular 1.x
  * Bootstrap 4.0.0-alpha.6
  * JQuery 3.x
    * JQuery-Validation
  * Font Awesome 4.x
